Palestra: Fundamentos de HTML e CSS - Construindo a Web do Zero

Nesta palestra introdutória, vamos explorar os fundamentos do HTML (HyperText Markup Language) e do CSS (Cascading Style Sheets), as duas principais linguagens utilizadas na criação de páginas web. A proposta é mostrar como sites são estruturados, estilizados e apresentados aos usuários. Durante o encontro, os participantes aprenderão:

  • Como funciona a estrutura básica de uma página HTML.
  • Como utilizar elementos como textos, imagens, links e listas.
  • Como aplicar estilos com CSS para transformar a aparência da página.
  • Conceitos de layout responsivo e boas práticas de codificação.
  • Demonstrações ao vivo para visualizar mudanças em tempo real.
